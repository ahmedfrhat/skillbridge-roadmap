كان الباكيند يرفض طلبات الاستدعاء نظراً لحظر كوكيز HttpOnly بين النطاقات المستضافة على Vercel و Azure. تم علاجها برمجياً بتحويل النظام لـ **Bearer JWT Flow**؛ حيث يتم حفظ التوكن في الـ localStorage تلقائياً وتمريره برأس ترويسة الطلب.

// كود التمرير والتعرف التلقائي بـ http.ts

if (typeof window !== "undefined") {
  const token = localStorage.getItem("skillbridge_token");
  if (token && !headers.has("Authorization")) {
    headers.set("Authorization", `Bearer ${token}`);
  }
}

3 ميزات إستراتيجية إضافية تجعل لجنة التحكيم تختارنا دون تردد

لكي نضمن الفوز الساحق، قمنا بصياغة 3 ميزات استباقية لم يتطرق إليها أي فريق منافس، تعزز الموثوقية وتدمج المنصة مع الحياة الأكاديمية والعملية بالكامل.

1. نظام اعتماد التدريب الميداني الجامعي (University Endorsement Loop)

تقوم الجامعات المصرية بطلب تدريب صيفي معتمد للطلاب كقيد للتخرج. قمنا بدمج دورة عمل مخصصة لأساتذة الجامعات (Academic Supervisors)؛ حيث يرسل الطالب مشروعه المنجز في المنصة للأستاذ لمراجعته، ليقوم الأستاذ باعتماده وتوثيقه كـ **ساعات تدريب صيفي معتمدة** برقم تسلسلي موحد من المنصة!

2. مولّد العقود الذكية بالذكاء الاصطناعي (AI SLA Contract Generator)

لحماية متبادلة للحقوق، قمنا بإعداد ميزة تقوم تلقائياً بصياغة عقد عمل قانوني مبسط باللغة العربية (SLA) يغطي: بنود الملكية الفكرية للكود، مواعيد التسليم والنسب، وشروط NDA الحساسة، بالاعتماد على ميزانية المشروع وحجم الفريق، ليوثق العقد بضغطة موافقة من الطرفين قبل الدفع.

3. لوحة الصدارة والأوسمة التنافسية (Gamified XP & Leaderboard)

تحفيز الطلاب على التعلم والتوثيق المستمر بربط الكويزات وحل المايكرو جيجز بنظام خبرة (XP) وشارات تظهر بملف الطالب (مثل: خبير كود نظيف، بطل السرعة، منقذ السيرفر). يتم فرز الطلاب في لوحة صدارة تظهر على الصفحة الرئيسية، مما يعزز المنافسة اليومية والنشاط.
